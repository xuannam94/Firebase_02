# **App Name**: PrimeMarket Pro

## Core Features:

- Corporate & Information Pages: Displays the company's introduction, core values, history, team, and contact information with a map placeholder.
- Product Browsing & Discovery: Features a responsive grid layout for products on the home and shop pages, including filtering by category/price, sorting, and search functionality.
- Product Detail View: Presents detailed product information, pricing, high-quality images, and an 'Add to Cart' button for individual items.
- Shopping Cart & Checkout: Manages selected products in a slide-over or dedicated page, calculates the total price, and facilitates order placement through a checkout form.
- Customer Inquiry Management: A dynamic contact form captures user inquiries and saves them to Firestore, with a protected admin view for message retrieval.
- Product Administration: A protected admin dashboard allows for adding, editing, and deleting product listings stored in Firestore.
- AI-Powered Product Description Tool: An integrated AI tool within the admin dashboard to generate concise and engaging product descriptions based on provided product attributes.

## Style Guidelines:

- Primary accent color: A professional and trustworthy blue (#397AAC), providing a calming and stable presence.
- Background color: A soft, almost white blue-grey (#ECF2F7), ensuring a clean, open, and professional aesthetic.
- Accent color: A vibrant teal-cyan (#1E93A9) for highlighting important calls to action and interactive elements, adding a modern touch.
- All text will use 'Inter', a grotesque-style sans-serif font known for its modern, clean, and highly legible characteristics, suitable for both headlines and body text to ensure a consistent corporate look.
- Use a set of minimalist line icons for navigation, shopping cart, and other interactive elements to maintain a clean and uncluttered interface consistent with the professional brand identity.
- Implement a responsive grid-based layout for product displays and a clear, structured format for corporate information pages, ensuring optimal readability and a modern visual hierarchy across all devices. Utilize full-width hero sections on key pages.
- Incorporate subtle hover effects on buttons and interactive components, along with smooth transitions for content loading and cart operations, to enhance user experience without detracting from professionalism.
